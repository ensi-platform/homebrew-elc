# homebrew-elc

This is the official Homebrew tap for [ELC](https://github.com/ensi-platform/elc). You can use it to install Bun on MacOS and Linux using Homebrew.

## Install
To install ELC with Homebrew, aka. brew, run the following commands:

```shell
brew tap ensi-platform/elc
brew install elc
```
If you want to install a specific release, you can include the version in the command. You can also view the list of releases here.
```shell
brew install elc@0.2.11
```
