# homebrew-elc

This is the official Homebrew tap for [ELC](https://github.com/ensi-platform/elc). You can use it to install ELC on macOS and Linux using Homebrew.

## Install
To install ELC with Homebrew, aka. brew, run the following commands:

```shell
brew tap ensi-platform/elc
brew install elc
```

This installs `v1.0.0`.

If you want to install a specific release, you can include the version in the command.

```shell
brew install elc@1.0.0
brew install elc@0.2.11
```
